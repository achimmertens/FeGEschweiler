Checkliste: Jährliche Datenaktualisierung

  1. 1. Neue Gewinn-Verlust-Berichte (gewinn-verlust-bericht_YYYY.csv) in /Daten ablegen
  2. 2. Neue Bilanzberichte (bilanzbericht_YYYY.csv) in /Daten ablegen
  3. 3. Budgets_YYYY.txt prüfen / aktualisieren und in /Daten ablegen
  4. 4. Falls vorhanden: Budget_YYYY.csv in /Daten ablegen (wird nach /Daten/result kopiert)
  5. 5. Prüfen: Kontoauszüge und Summen-Salden in /Daten oder /old ablegen
  6. 6. node generate-presentation.js ausführen und debug.log lesen
  7. 7. Ergebnis prüfen: Daten/result enthält Budget_YYYY.html, Ausgaben_YYYY.html, Einnahmen_YYYY.html, Entwicklung_YYYY.html, Präsentation.pptx
  8. 8. PPTX öffnen und Folien prüfen; ggf. manuelle Korrekturen an Budget-Planungen vornehmen
  9. 9. Website/Index prüfen: /Daten/result/index.html öffnet alle Seiten
  10. 10. Repository commit & push: git add -A && git commit -m "update data YYYY" && git push
